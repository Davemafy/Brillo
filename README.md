# <img src="./public/brillo.svg" alt="Brillo logo"  />

Easy to use learning tracker that helps you log daily learning activity and stay consistent.

## Features
- Add daily learning entries
- View learning history
- Responsive, mobile-first design

## Tech Stack
- React
- Typescript
- Tanstack Router
- Tailwind CSS
- Node.js

## Why I built this
I wanted to practice building a real, usable web app instead of isolated components.

## Status
MVP in progress – core features implemented.
